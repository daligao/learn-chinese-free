## Which tool to start with

**Beginner (HSK1–3):** Start with Mandarin Flashcards to build vocabulary. Use the Pinyin Annotator to read anything above your level.

**Intermediate (HSK4–6):** Chinese Reading Lab and Chengyu Stories will do more for comprehension than drilling vocabulary. Use the Writing Toolkit for exam prep.

**Any level:** Pinyin Annotator works on any text — paste from WeChat, news, subtitles, or textbooks.
