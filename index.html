Pinyin Annotator
拼音标注器
All levels Reading aid 2,500+ chars
Paste any Chinese text — pinyin appears above every character instantly. Four-tone color coding (red/orange/green/purple), HSK level highlight, click any character for details. The fastest way to read Chinese you don't know yet.
Try It Free → ⭐ GitHub

Chinese Reading Lab
历史阅读理解
HSK 4–6 Reading 10 stories
10 real historical decisions told in Chinese. Mandela in prison, Jobs getting fired, Musk betting everything on Tesla. Make the decision before you see the result, then answer 3 comprehension questions. Click any word for instant pinyin + meaning popup.
Start Reading → ⭐ GitHub
🐉
Chengyu Stories
成语故事
HSK 4–6 Idioms 20 stories
20 classic 成语 — 4-character idioms used constantly in modern Chinese. Learn each idiom through its 2,000-year-old origin story in Chinese (with pinyin popup), then test yourself: given a modern real-world scenario, pick the right idiom from 4 options.
Learn Idioms → ⭐ GitHub
🃏
Mandarin Flashcards
汉语词汇卡片
HSK 1–3 Vocabulary 400+ words
400+ HSK1–3 words with spaced-repetition flashcards. Filter by level, click characters for pinyin popup. Includes a story reader — English paragraph first, tap to reveal the Chinese. Good for learning the 1,200 characters you need for daily conversation.
Start Drilling → ⭐ GitHub
✍️
Chinese Writing Toolkit
汉语写作工具
HSK 4–6 Writing 11 types
Model essays and sentence banks for 11 types of Chinese writing: invitation letters, thank-you notes, complaint letters, cover letters, self-introduction, and more. 13 grammar patterns, a vocabulary upgrade table, and a practice pad. Built for the HSK writing section.

An ordinary guy living in China

I'm Tim. I live in China, speak Chinese daily, and build tools I wish existed when I was learning. These aren't sponsored or affiliated with any app — they're just free HTML files that do one thing well. I write about what I build at ordinarymantrying.com. If a tool helped you, a ⭐ on GitHub helps other learners find it.

All tools are free, offline, and open source. No tracking cookies, no accounts, no paywalls.
